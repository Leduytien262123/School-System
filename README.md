## 技术选型

Vue-Bag-Admin sử dụng các công nghệ mới nhất như Vue3, Vite5, Naive UI, JavaScript, Pinia... Phát triển frontend hiện đại, tích hợp qua npm, cho phép mở rộng tính năng mà không cần sửa mã nguồn gốc, giúp ứng dụng module hóa, dễ bảo trì. Giao diện đẹp, đáp ứng tốt cho hệ thống quản trị.

## Đặc điểm nổi bật

#### Phát triển responsive

Giao diện xây dựng với tailwindcss, tự động thích ứng PC, mobile, tăng trải nghiệm người dùng, nhiều chủ đề giao diện, đẹp mắt, giảm chi phí phát triển và bảo trì.

#### Dễ sử dụng

Cài đặt qua npm hoặc tải mã nguồn để phát triển, tách biệt framework (packages) và ứng dụng (app), giảm phụ thuộc giữa các dự án, tăng khả năng mở rộng.

#### Cộng đồng mạnh mẽ

Sử dụng các công nghệ phổ biến như Vue3, Vite5, Naive-ui, Pinia, Strapi5, MySQL... Miễn phí, thương mại hóa thoải mái, không lo giới hạn công nghệ.

#### Dịch vụ API

Backend sử dụng Strapi (Node.js), CMS mã nguồn mở, cho phép tùy biến API, hỗ trợ GraphQL, phân quyền chi tiết, xác thực linh hoạt.

#### Đầy đủ tính năng

Quản lý quyền, đa ngôn ngữ, cache trang, đổi chủ đề, hiệu ứng chuyển trang, layout đa dạng, route động, quản lý trạng thái, micro frontend, đóng gói request, hơn 20 template trang mẫu... Giúp bạn phát triển hệ thống quản trị nhanh nhất, giảm khối lượng công việc.

#### Phát triển dạng plugin

Framework hỗ trợ phát triển dạng plugin, mở rộng tính năng mà không cần sửa mã nguồn, tăng tính module hóa, dễ bảo trì, phù hợp với các dự án cần mở rộng và tùy biến cao.

## 安装

```bash
pnpm i vue-bag-admin --save
```

## 快速开始

```javascript
import install from "vue-bag-admin";

const { app } = install();
app.mount("#app");
```

## 源码开发

- `main`为最新分支，包含所有源代码，前端和后端程序
- `v4` 为上一个版本分支，包含所有源代码，前端和后端程序(停止维护)
