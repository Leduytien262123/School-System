# Mở rộng toàn cục

Bên trong framework đã mở rộng sẵn một số thuộc tính, bạn có thể sử dụng trực tiếp:

```javascript
const ctx = new Framework({
  app, // Đối tượng vue
  pina, // Quản lý trạng thái pina
  helpers, // Các hàm tiện ích
  radash, // Thư viện tiện ích frontend https://www.radash.wiki/
  hooksPlus, // vue-hooks-plus https://inhiblabcore.github.io/docs/hooks/
  dayjs, // Thư viện xử lý thời gian
  http, // Gói gọi API https://alova.js.org/
  nprogress, // Thanh tiến trình
});
```

## [Trang chủ Radash](https://www.radash.wiki/)

Radash là thư viện tiện ích frontend tốt nhất, không phụ thuộc, mô-đun hóa, hiệu năng cao cho JS/TS.

## [Trang chủ HooksPlus](https://inhiblabcore.github.io/docs/hooks/)

Cung cấp các hook cơ bản và nâng cao, đóng gói logic hiệu năng cao, đáp ứng nhiều tình huống, cú pháp đơn giản, dễ sử dụng, tài liệu chi tiết.

## [Trang chủ Alova](https://alova.js.org/)

Công cụ gọi API thế hệ mới, đơn giản hóa tối đa quy trình tích hợp API, chỉ cần một bước.

## [Trang chủ Day.js](https://day.js.org/zh-CN/)

Day.js là thư viện JavaScript nhẹ để xử lý thời gian và ngày tháng, API tương tự Moment.js.
